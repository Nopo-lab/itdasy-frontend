## 1. 디지털 친숙도별 병목

### 1.1 친숙도 "하" (10명, 20%)
**페르소나**: P03 이미경, P06 한선영, P12 이주연, P14 장미라, P21 권나래, P24 남영주, P34 장하연, P39 주민아, P46 고은영, P50 최영지
- 평균 연령 48.6세 / 평균 영업기간 16.5년 / 평균 매출 9.7M
- 단골 비율 매우 높음 (평균 0.84) — 단골 관리 필수

#### 핵심 병목 (P0)
1. **카카오 OAuth 콜백 끊김** — 카카오 앱 → 잇데이 자동 복귀 안 되면 어디로 갔는지 모름. 중복 가입 위험 (P03·P22 시나리오)
2. **회원가입 단계 너무 많음** — 약관 동의 글자 작음, 사업자 인증 서류 사진 업로드 부담. 도중 포기 위험 (P14·P34·P46 공통)
3. **음성 입력 마이크 권한 거부 후 복구 모름** — 한 번 거부하면 어떻게 다시 켜는지 미가이드 (P03·P14·P24·P42·P50)
4. **사투리/억양 인식 저조** — 전남 목포(P46), 강원 춘천(P34), 전북 전주(P14) 사용자 STT 인식률이 표준어 대비 떨어질 가능성
5. **글자 크기 접근성 옵션 미발견** — 메뉴 깊숙이 → 못 찾음. 가입 폼 자체가 글자 작음 (P14·P21·P24·P39·P46)
6. **종이 노트 → 앱 일괄 import 부재** — 100~410명 단골을 수기로 추가? 도중 앱 종료 시 폼 자동 복구 작동 미신뢰 (P03·P21·P24·P34·P46·P50)

#### 보조 병목 (P1)
- 카톡 채널 미연결 시 예약 알림 무발송 — 본인 인지 부재 (P22)
- 사업자 인증 서류 업로드 — 문서 사진 vs 갤러리 차이 모름
- 도움 챗봇 응답 너무 일반적 — 본인 문제 못 풀음 (P14·P34·P46)
- 카드사 영수증 포맷 다양 — 일부 인식 실패 (P11·P24)

### 1.2 친숙도 "중" (25명, 50%)
**페르소나**: P01·P04·P07·P09·P11·P15·P17·P19·P20·P22·P26·P27·P29·P31·P33·P36·P38·P41·P42·P44·P47·P48 외
- 평균 연령 36세 / 평균 영업기간 6.5년 / 평균 매출 6.0M
- 가장 큰 그룹 — 출시 후 주요 사용자

#### 핵심 병목 (P0)
1. **옵티미스틱 UI 작동했지만 사용자 인지 못함** — 매출 추가 후 대시보드 즉시 반영 안 보이면 한 번 더 누름 → 이중 등록 (P01·P11·P20)
2. **인스타 발행 진행률·카운트다운 부재** — "발행 중" 상태 → 멈췄나 의심 → 다시 누름 → 중복 발행 위험 (P01·P26)
3. **CSV 한글 깨짐 의심** — 세무사·엑셀 호환 OK인지 안내 부재. UTF-8 BOM 처리 명시 필요 (P01·P04·P29·P38·P44·P50)
4. **카톡 일괄 발송 한도·차단 위험 안내 부재** — 100명 일괄 보내면 카카오 정책상 차단당할 수 있음 (P04·P28)
5. **단골 안부 메시지 톤이 본인 말투 아님** — 페르소나 학습 안 된 상태에선 일반 톤 → "내 어조 아닌데" 거부감 (P01·P19·P22·P29·P38·P44)
6. **다중 사진 업로드 진행률 부재** — 영수증 5~50장 일괄 업로드 시 진행 상태 시각 표시 없음 → 멈췄나 의심 → 추가 업로드 → 중복 (P01·P07·P24)
7. **카톡 채널 미연결 → 예약 알림 무발송 — 사용자 인지 부재** — 발송 사실 본인 확인 어려움 (P15·P22·P48)
8. **재고 임계치·유통기한 의미 직관 부재** — 임계 5개 = 재고 5 이하 알림? 5 이상? 헷갈림 (P04·P48)

#### 보조 병목 (P1)
- 노쇼 누적 자동 차단 기준 불명 (P02·P07)
- 멤버십 정책 변경 시 기존 누적 회수 백필 부재 (P02·P17·P21·P29·P39·P41)
- 패키지 vs 1회 결제 자동 분리 부재 (P04·P33)
- 의료광고법 위반 카피 자동 생성 우려 (피부·반영구) (P04·P16·P25·P48)
- 음성 입력 노이즈 환경 인식 저조 (P22·P29)
- 일괄 편집 (예약 변경 폭주) 부재 (P22·P28·P44)

### 1.3 친숙도 "상" (15명, 30%)
**페르소나**: P02·P05·P08·P10·P13·P16·P18·P23·P25·P28·P30·P32·P35·P37·P40·P43·P45·P49 (*overlap는 일부 dual)
- 평균 연령 28세 / 평균 영업기간 3.8년 / 평균 매출 6.4M
- 인스타 팔로워 평균 5,800 — 마케팅 활용도 높음

#### 핵심 병목 (P0)
1. **키보드 단축키 미지원** — Cmd+S(저장), Cmd+Enter(발행), Cmd+K(검색) 등 미지원. MacBook·iPad+키보드 사용자 답답함 (P35·P37·P45·P49)
2. **멀티 디바이스 sync 5분 idle 후만 작동** — iPhone ↔ iPad ↔ MacBook 즉시 sync 부재. 지금 폰에서 입력 → iPad에서 안 보임 (P02·P08·P30·P37·P49)
3. **MacBook 웹 미지원** — PWA가 데스크톱 브라우저에서 풀 동작 안 한다면 MacBook Pro 사용자 (P30·P49) 매우 불만
4. **스레드/틱톡/유튜브쇼츠 직접 발행 미지원** — 캡션 생성은 되지만 발행은 인스타만 → 멀티 플랫폼 사용자 (P02·P05·P10·P28·P35·P49) 두 번 일함
5. **페르소나 학습 시드 효과 측정·가이드 부재** — 시드 5개 vs 10개 차이 안내 없음. 시드 가이드 모범 예시 부재 (P02·P05·P13·P18·P23·P25·P32·P35·P40·P43·P45)
6. **스레드 글자수 자동 fit 부재** — 500자 제한 자동 잘림 안 함 → 직접 편집 (P02·P28·P45)
7. **데이터 내보내기 필터 단순** — 직원별/카테고리별/성별/태그별 필터 부재. 분석가 사용자 (P08·P30·P37·P41·P45·P49) 답답
8. **인스타 인사이트·광고 효과 측정 연동 부재** — 광고비 대비 효과 매칭 수동. 디지털 능숙자에게 핵심 페인 (P08)

#### 보조 병목 (P1)
- 갤러리 자동 태깅 부재 — 컨셉별 분류 수동 (P13·P26·P32)
- 9:16 스토리 BGM 자동 추가 부재 (P23·P43)
- 비포애프터 슬라이드 효과 옵션 부족 (P16·P40)
- 해시태그 본인 큐레이션 미세조정 부재 (P11·P35·P40)

## 3. 신규(<3년) vs 베테랑(10년+) 병목

### 3.1 신규 (영업 1~3년, 12명)
- **매출 적자 직전 — UI 톤이 압박 주면 안 됨** (P10·P32 — 적자 직전)
- 신규 손님 SNS 광고 어떻게 만들지 막막 (P05·P23·P32·P43)
- 페르소나 시드 시간 투자 가치 의심 (P10·P32 자원 부족)

### 3.2 베테랑 (10년+, 14명)
- **종이 노트 → 앱 일괄 import 부재** (P03·P12·P14·P21·P24·P34·P39·P46)
- 18~22년치 누적 데이터 import 부재 → 분석 의미 떨어짐
- 디지털 거부감 + 단골 비율 0.8+ → 멤버십 백필 절실

## 5. 출시 전 검증 우선순위 (P0)

### Tier-1 (출시 블로커)
1. **카카오 OAuth 콜백 안정성** — 친숙도 하 그룹 가입 여부를 결정
2. **CSV 한글 인코딩 (UTF-8 BOM)** — 세무사 전송 실패 시 환불 클레임 위험
3. **이중 등록 방지 (옵티미스틱 UI 피드백 강화)** — 매출 1번만 + 명확한 success 피드백
4. **마이크 권한 거부 후 복구 가이드** — Settings 딥링크 또는 인앱 가이드
5. **다중 사진 업로드 진행률 표시** — 영수증 5~50장 batch 시 시각 progress
6. **단골 일괄 import (CSV 또는 카카오 채널)** — 베테랑 10년+ 단골 백필 통로 필수
7. **카톡 알림 발송 결과 인지 UI** — "발송 성공/실패" 명확 표시
8. **인스타 발행 진행률 카운트다운**

### Tier-2 (UX 만족도)
9. 페르소나 학습 시드 가이드 + 효과 측정
10. 멀티 디바이스 즉시 sync (현재 5분 idle)
11. 데이터 내보내기 필터 (직원·카테고리·성별·태그)
12. 의료광고법·성인 콘텐츠 가드레일 (피부·반영구·왁싱)
13. 음성 사투리 인식 강화 (한국어 STT 모델 개선)
14. 글자 크게 보기 접근성 — 가입 폼부터 적용
15. 멤버십 정책 변경 시 누적 회수 백필
16. 일괄 편집 (예약 다중 수정)

### Tier-3 (Pro 사용자 만족)
17. 키보드 단축키 (Cmd+S·Cmd+Enter·Cmd+K)
18. MacBook 웹 PWA 풀 지원
19. 스레드·틱톡·유튜브 쇼츠 직접 발행
20. 갤러리 자동 태깅
21. 9:16 BGM 자동
22. 인스타 인사이트 연동

## 6. 시나리오 적중률 가설

각 친숙도 그룹이 가장 높은 빈도로 시도할 시나리오 톱5:

| 친숙도 | Top Scenario (예상 빈도) |
|---|---|
| **하** | ① 음성 매출 → ② 단골 등록 (수기) → ③ 가입 → ④ 예약 추가 (전화) → ⑤ 글자 크게 |
| **중** | ① 영수증 챗봇 → ② 1초 캡션 → ③ 단골 안부 → ④ 예약 추가 → ⑤ 데이터 내보내기 |
| **상** | ① 1초 캡션 + 9:16 → ② 멀티 발행 → ③ 페르소나 학습 → ④ 멀티 디바이스 sync → ⑤ 데이터 분석 |

## 7. 후속 액션 제안

- 위 P0 8개에 대해 자동화 E2E 테스트 작성 (Playwright + 백엔드 통합)
- 친숙도 "하" 그룹 대상 베타 테스터 5명 (40대 후반~50대) 모집 — 가입~첫 매출 등록 시간 측정
- 1초 캡션 의료광고법·성인 콘텐츠 가드레일 LLM 시스템 프롬프트 강화
- CSV export UTF-8 BOM 보장 + 엑셀 호환 안내 토스트
- 단골 일괄 import: CSV 템플릿 다운로드 + 업로드 + 매핑 마법사 기획
